# My Moodboard — Replacement Instructions

## Goal
Replace the current **Moodboard** section inside the existing Wardrobe page with a more expressive, editorial, free-flowing moodboard experience.

This is **not** a new product page.
This is a **direct replacement** for the current Moodboard section in the same Wardrobe feature.

The new version should feel like a beautiful, tactile, Pinterest-inspired moodboard that still belongs to the same brand as the Wardrobe page.

---

## What stays the same
Keep the existing data and behaviors already in the codebase:
- boards
- pins
- pin types: product, look, note, color
- create / delete board
- addPin, removePin, movePin
- existing brand styling language

Do **not** change the underlying feature model. Only replace the presentation and interaction feel.

---

## What must be removed
Remove the current rigid, tool-like layout from the Moodboard section.

The current section is structured like a split workspace with a board list on the left and a grid canvas on the right, using a strict 8-column layout and control-heavy pin actions fileciteturn6file1.

Remove:
- the left-side “Board list” panel
- the right-side “Moodboard canvas” panel feel
- the strict grid-editor look
- visible grid borders / dashed canvas treatment
- heavy button rows for adding pins at the top
- anything that feels like a layout editor, CMS, or design tool

---

## What must replace it
Replace the current layout with a **single, immersive Moodboard experience**.

The page should feel more like:
- a pinboard
- a collage surface
- a creative style wall
- a tactile mood capture space

This should be visually rich, but not chaotic.

---

## Page structure
### 1) Header
At the top, show:
- small brand label
- page title: **Moodboard**
- subtitle: **Explore. Play. Refine your style story.**

The header should feel spacious, elegant, and editorial.

### 2) Main moodboard surface
Below the header, show a large free-form composition area.

This is the main transformation.

The canvas should feel like a real moodboard made from mixed references and saved style pieces, not a rigid grid.

Use:
- varied card sizes
- layered images
- subtle overlap
- slight rotation on a few pieces, if tasteful
- soft shadows
- paper-like depth
- quiet, premium neutrals

The arrangement should feel curated, not mechanical.

### 3) Minimal board switching
Board switching should not dominate the layout.

Use a small, quiet control such as:
- a compact board selector near the top
- or a minimal dropdown
- or a subtle tab/pill control

Do **not** bring back a heavy left-side panel.

### 4) Add content interaction
Adding new pins should feel lightweight and secondary.

Use one minimal trigger such as:
- a floating add button
- a small plus button near the header
- a compact quick-add menu

Do not place large add buttons across the layout.

---

## Visual direction
Use the uploaded reference images as the tone guide:
- warm neutrals
- cream, beige, soft browns
- soft editorial typography
- smooth rounded cards where appropriate
- delicate shadows
- calm luxury
- tactile, magazine-like layering

The moodboard should feel more expressive than the wardrobe page, but still clearly part of the same family.

---

## Composition style
The moodboard should feel intentionally imperfect in a beautiful way.

Use a mix of content types and sizes:
- product images
- full looks
- texture swatches
- color blocks
- handwritten-style notes
- fabric or material references
- small editorial snippets

The layout should feel like a visual story, not a structured catalog.

Avoid uniformity.
Avoid a repetitive grid.
Avoid perfect alignment everywhere.

---

## Pin treatment by type
### Product pin
- real product image
- soft background
- rounded card or paper-frame treatment
- subtle shadow

### Look pin
- larger than a product pin
- can use the existing look card treatment
- should feel like a reference anchor in the board

### Note pin
- styled like a paper note or sticky note
- minimal handwritten or serif-style feel
- light, expressive, personal

### Color pin
- simple color chip or swatch block
- minimal label
- small but important

---

## Interaction feel
The page should feel playful and exploratory.

The user should be able to:
- browse boards
- add visual references
- move pins around
- remove pins
- build an aesthetic story

But the main surface should never feel like a technical editor.

The editing actions should be quiet and secondary.

---

## What not to do
Do not make this feel like:
- Figma
- Canva
- a grid builder
- a dashboard
- a spreadsheet
- a CMS
- ecommerce browsing

Do not show:
- rigid grid lines
- dense control bars
- heavy form fields
- overwhelming tool chrome
- marketplace-style product cards

---

## Moodboard personality
This page should feel like:
- collecting inspiration
- building a style story
- pinning atmosphere and references
- creating a visual direction for a future look

It should feel more expressive and more open than the wardrobe page.

The wardrobe page is more composed.
This moodboard page is more fluid.

---

## Relationship to the Wardrobe section
The two sections should feel related, but not redundant.

### My Wardrobe
- structured
- curated
- outfit-focused
- gallery-like

### Moodboard
- freer
- more expressive
- reference-driven
- collage-like

The moodboard should feel like the place where ideas are gathered before they become wardrobe compositions.

---

## Acceptance checklist
The replacement is correct only if:
- the old split board list + canvas layout is gone
- the page feels immersive and editorial
- the moodboard surface feels freeform, not grid-locked
- the page uses the same underlying pin logic
- the visual tone matches the reference images
- the result feels creative, tactile, and premium
- the page still clearly belongs to the same Wardrobe product
